# 🎓 Student Registration System

A simple  **Student Registration System** developed using **Python**. This project allows users to manage student records through a menu-driven interface. Student data is stored permanently in a JSON file.

## 📌 Features

- ➕ Add a new student
- 📋 View all students
- 🔍 Search a student by ID
- ✏️ Update student details
- 🗑️ Delete a student
- 💾 Save data using a JSON file
- 📂 Automatically load saved data when the program starts
- ⚠️ Prevent duplicate student IDs
- ✅ Handle invalid user input using exception handling



## 🛠️ Technologies Used

- Python
- Object-Oriented Programming (OOP)
- JSON
- File Handling
- Exception Handling

1. Install Python 3.

3. Open a terminal in the project folder.
4. Run the program:

